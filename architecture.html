Proposed Architecture

Production-grade from
day one.

Mobile-first is an architectural decision, not a CSS afterthought

Building mobile-first means every design decision — layout, navigation, tap target size, form inputs, loading states — starts from a 375px phone screen and scales up. It's fundamentally different from building a desktop app and adding media queries. The result is a platform that feels native on a phone and equally polished on a desktop.

For Orr Group's target market — small nonprofit staff running lean teams — this is the difference between a platform they use every day and one they abandon after two weeks. Your clients will encounter it first on a phone. The first impression has to be excellent.

Security & compliance built in

✓

Data encryption at rest and in transit

Aurora storage encrypted with AWS KMS. All API traffic over TLS 1.3. S3 objects server-side encrypted.

✓

Complete audit trail

Every user action, every API call, every data modification logged with timestamp and user ID. CloudTrail captures infrastructure events.

✓

Tenant data isolation

Row-level security on Aurora. Organization A cannot query Organization B's data under any circumstances — enforced at the database layer, not just the application layer.

✓

GDPR & CCPA ready

Data portability guaranteed — every user can export their data. Right-to-deletion supported. Data residency options available for organizations with specific requirements.

✓

No AI training on client data

The Claude API processes queries in real time. Client data is never used to train or fine-tune the underlying model. Anthropic's enterprise privacy terms apply.

✓

AWS BAA available (Option 3)

Under the licensed SaaS model, 2516 manages all AWS compliance configurations. Business Associate Agreements available if any client data touches health information.
