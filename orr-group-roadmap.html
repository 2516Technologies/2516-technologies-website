Phased Roadmap

Four phases. Each one building on the last. Nothing ships until it's ready.

The platform is built in phases — stable foundation first, intelligence second, analytics third, advanced AI fourth. Your approval is required at every milestone checkpoint before the next phase begins.

Phase 1A — Months 1–3
Infrastructure & Foundation
The backbone everything else runs on

This phase builds the infrastructure that makes every other feature possible. Nothing in Phase 2, 3, or 4 can exist without this being solid.

AWS environment setup — VPC, IAM roles, environments (dev/staging/prod)
Aurora PostgreSQL Serverless v2 — schema, indexes, row-level security
Amazon S3 — file storage with presigned URL access
AWS Lambda + API Gateway — serverless API layer
Amazon Cognito — user auth, roles, MFA
Multi-tenant architecture — org isolation enforced at database level
Stripe integration — subscription billing, plan management, webhooks
CI/CD pipeline — GitHub Actions automated test + deploy
Full audit trail — every action timestamped and logged
Phase 1B — Months 3–5
Product & Interface
The platform CJ's clients actually use

Built on the Phase 1A foundation, this phase delivers the user-facing product — mobile-first, polished, and ready for 1,000 organizations on launch day.

Mobile-first React/Next.js frontend — designed for 375px screens first
CRM module — contact management, org records, interaction history
User dashboard — onboarding flow, progress tracking, account management
Email notifications via SES — transactional + workflow-triggered
SMS notifications via SNS — high-priority alerts and reminders
Admin panel — user management, subscription oversight, platform analytics
QA and cross-device testing — phones, tablets, desktop browsers
Staging and production deployment — ready for beta users

Milestone checkpoint: Phase 1B completes with a demo of the full platform to CJ. Your approval is required before beta launch. 2516 does not launch to users without sign-off.

Phase 2 — Months 6–9
Separate contract
Intelligence — the AI features

Phase 2 builds the AI surfaces that differentiate the platform. It begins after Phase 1 is stable in production — rushing AI features onto an unstable foundation is how you get a platform that looks bad and doesn't work.

Virtual consultant — Claude-powered advisor trained on Orr Group methodology
RAG pipeline — document ingestion, embedding, retrieval-augmented generation
Claude-powered chatbot — document Q&A, platform help, contextual assistance
Interactive avatar — human-feeling engagement interface
Conversation history and context management

Phase 2 is a separate contract scoped and priced after Phase 1 is live. Claude API costs (~$38–$58/mo at 1,000 orgs) will be added to CJ's AWS bill at this stage.
